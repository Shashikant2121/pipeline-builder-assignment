# Pipeline Builder (DAG Validation)

A visual pipeline builder built using ReactFlow with backend DAG validation.

## Tech Stack
- Frontend: React + Vite + ReactFlow
- Backend: Node.js + Express

## Features
- Drag & connect pipeline nodes
- Custom node types (Input, Text, Output)
- Backend DAG validation
- Node and edge count API

## How to Run

### Backend
```bash
cd backend
npm install
npm run dev

Frontend
npm install
npm run dev
